What a waste

For this project I had to make a game based on an already existing concept for a trash recycling company. The concept was a puzzle game where you have to sort the right type of trash int he same type of container. You have got a time limit to do it. You can move the arrows on the field to redirect the trash. If you collect a type of trash in the incorrect container it gives you a strike. You lose at 3 strikes or when the time is up.

As for the mechanics in this game I have included: Redirection of trash on the lanes, the trash stays in a lane, score, strike system and a timers.

I made one complete level for this project.
